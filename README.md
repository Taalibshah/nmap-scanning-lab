# Nmap Scanning Lab

## Overview
This project demonstrates various Nmap scanning techniques performed in a Kali Linux virtual machine.

## Scans Performed

- TCP SYN Scan (-sS)
- TCP Connect Scan (-sT)
- UDP Scan (-sU)
- NULL Scan (-sN)
- FIN Scan (-sF)
- ACK Scan (-sA)
- Xmas Scan (-sX)
- Host Discovery (-sn)
- Timing Templates (-T0 to -T5)

## Lab Environment

- Kali Linux
- Nmap
- Virtual Machine
